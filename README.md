# 📚 42 Piscine C — Guías de Estudio y Repaso

Repositorio de materiales de repaso para la **Piscine de C de 42**. Todos los archivos son páginas HTML interactivas pensadas para estudiar directamente en el navegador, sin necesidad de instalar nada.

> 💡 **Los enlaces "Ver en navegador" funcionan una vez hayas activado GitHub Pages en tu repositorio.**
> Ve a `Settings → Pages → Source → main → / (root)` y guarda.

---

## 📂 Archivos

### 🟢 `piscine42_cheatsheet.html`
**Cheat Sheet general de la Piscine**

Referencia rápida de todas las funciones básicas de C organizadas por categoría (imprimir, strings, números, matemáticas). Para cada función incluye:
- Prototipo y descripción
- Explicación paso a paso de la lógica
- Código completo
- Casos borde importantes

Funciones cubiertas: `ft_putchar`, `ft_putstr`, `ft_putnbr`, `ft_strlen`, `ft_strcpy` y muchas más.

🌐 [Ver en navegador](https://frangomeznavas.github.io/42school_C_resources/piscine42_cheatsheet.html) · 📄 [Ver archivo](https://github.com/franGomezNavas/42school_C_resources/blob/main/piscine42_cheatsheet.html)

### 🟡 `exam42_cheatsheet.html`
**Cheat Sheet + Checklist para el Exam Rank 02**

Guía de preparación para el examen con checklist interactivo (puedes marcar los ítems). Contiene:
- **Flujo mental** paso a paso para afrontar cada ejercicio del examen
- **Conceptos clave**: `write()`, `argc`/`argv`, iteración de strings, malloc...
- **Fórmulas y patrones** de código más frecuentes
- **Trampas comunes** a evitar
- **Man pages** disponibles durante el examen

🌐 [Ver en navegador](https://frangomeznavas.github.io/42school_C_resources/exam42_cheatsheet.html) · 📄 [Ver archivo](https://github.com/franGomezNavas/42school_C_resources/blob/main/exam42_cheatsheet.html)

### 🟣 `guia_piscine_c.html`
**Guía completa de la Piscine C (C00 → C03)**

Guía de estudio que cubre los primeros módulos de la Piscine con plan de estudio de 7 días. Incluye:
- **La Norma** — reglas críticas que debes cumplir
- **C00** — `write`, loops, ASCII
- **C01** — Punteros
- **C02** — Strings y manipulación de caracteres
- **C03** — Comparación y concatenación de strings

## 🚀 Cómo activar GitHub Pages

Para que los enlaces "Ver en navegador" funcionen:

1. Ve a tu repositorio en GitHub
2. Entra en **Settings → Pages**
3. En **Source** selecciona `Deploy from a branch`
4. Elige la rama `main` y la carpeta `/ (root)`
5. Guarda — en unos minutos los enlaces estarán activos
